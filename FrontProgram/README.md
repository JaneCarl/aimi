# 艾米格斗少儿培训机构 - 后台管理系统前端

## 技术栈
- Vue 3
- Element Plus
- Pinia
- Vue Router 4
- Axios
- ECharts

## 安装步骤

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 默认账号
- 用户名: admin
- 密码: admin123

## 功能模块
- 数据看板：显示关键统计数据和图表
- 教练管理：教练的增删改查
- 学员管理：学员的增删改查、课时充值
- 课程管理：课程和班级的管理
- 请假审批：审批家长的请假申请
- 财务管理：财务记录的增删查
- 内容管理：轮播图和优惠课程的管理

## 项目结构
```
src/
├── api/              # API接口
├── layouts/          # 布局组件
├── router/           # 路由配置
├── stores/           # Pinia状态管理
├── styles/           # 全局样式
├── utils/            # 工具函数
├── views/            # 页面组件
├── App.vue           # 根组件
└── main.js           # 入口文件
```
