# 艾米格斗少儿培训机构 - 微信小程序

## 项目说明
这是艾米格斗少儿培训机构的微信小程序端，包含家长端和教练端两个角色。

## 功能模块

### 家长端
1. **首页**
   - 轮播图显示机构活动和公告
   - 宝宝照片展示
   - 下一次课程信息
   - 优惠课程列表

2. **课表**
   - 日历视图显示课程安排
   - 课程列表展示
   - 请假功能

3. **成长**
   - 纵向滚动查看成长轨迹
   - 教练评语和照片/视频

4. **我的**
   - 学员信息展示
   - 切换孩子功能
   - 课消明细
   - 退出登录

### 教练端
1. **首页**
   - 轮播图显示机构活动和公告
   - 教练信息
   - 今日课程简览

2. **课堂**
   - 今日课程列表
   - 课程状态显示（已学/正在上/未开始）

3. **上课详情**
   - 学员签到（正常/迟到/请假/缺席）
   - 打分和评价
   - 上传训练视频或图片
   - 积分发放

4. **学生**
   - 所有学生列表
   - 学生详情（档案、最近课堂记录）

5. **我的**
   - 个人信息
   - 教学统计
   - 消息通知
   - 退出登录

## 技术特点
- 自定义TabBar，根据角色显示不同标签
- 微信登录（openid方式）
- 模块化API请求封装
- 统一的错误处理
- 精简的模拟数据

## 开发说明
1. 使用微信开发者工具打开项目
2. 修改 `project.config.json` 中的 `appid` 为你的小程序AppID
3. 配置服务器域名（在开发工具中勾选"不校验合法域名"）
4. 登录时使用测试OpenID（开发阶段）

```
├── login/              # 登录页
├── parent/             # 家长端页面
│   ├── home/          # 首页
│   ├── schedule/      # 课表
│   ├── leave/         # 请假
│   ├── growth/        # 成长
│   └── my/            # 我的
├── coach/              # 教练端页面
│   ├── home/          # 首页
│   ├── class/         # 课堂
│   ├── detail/        # 上课详情
│   ├── students/      # 学生列表
│   ├── student-detail/# 学生详情
│   └── my/            # 我的
custom-tab-bar/         # 自定义TabBar
utils/                  # 工具类
images/                 # 图片资源
app.js                  # 小程序入口
app.json                # 小程序配置
app.wxss                # 全局样式
```

## 注意事项
1. 所有图片均使用占位符或本地图片
2. 不涉及微信支付功能
3. 不使用小程序云开发
4. 保持代码精简，避免冗余
